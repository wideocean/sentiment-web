Sentiment Detection DEMO Request Handling

Sentiment Servlet

Request URL:

/sentiment-web/detectSentiment?reviewtext=&keywords=false

Parameters

- reviewtext (String)
The text to be analysed

- keywords (true/false)
true = overall sentiment + all detected sentiment words including their scores
false = overall sentiment
